Woodworking 101 (In-Person)
Use our woodshop to build a sturdy plywood box
Use Hacksburg's miter saw, drill press, and jigsaw to construct a sturdy plywood box with handles and a lid. A great introduction to woodworking and the tools that Hacksburg offers.

This class is for attendees age 16 and up. Minors will need to have parent/guardian supervision for the entire duration of the class, and parent/guardian must approve all shop tool use.

Time: November 14th from 1:00 to 4:00pm
Location: Online and in person at Hacksburg; 1872 Pratt Drive Suite 1620.
Enter through the covered double doors at the back of the building.
URL: meet.hacksburg.org/class
Cost: $10 for Hacksburg members; $15 for non-members. Online payments accepted at paypal.me/hacksburg.

Build Your Own Website
No coding experience required
Learn the basics of HTML and CSS to build a simple personal website. Bring a laptop loaded with your favorite text editor (we recommend VS Code), and be sure to sign up for a GitHub account before class (we'll host our sites using GitHub pages). Looking for some inspiration? Check out NeoCities.
